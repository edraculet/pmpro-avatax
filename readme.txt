== Installation ==

= Prerequisites =
1. You must have Paid Memberships Pro installed and activated on your site. In addition, you will also need a subscription with Avalara (http://avalara.com)

= Download, Install and Activate! =
1. Download the latest version of the plugin.
1. Unzip the downloaded file to your computer.
1. Upload the /pmpro-avalara/ directory to the /wp-content/plugins/ directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
